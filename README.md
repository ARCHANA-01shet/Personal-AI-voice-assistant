# Personal-AI-voice-assistant
Personal-AI-voice-assistant
Project Description
This project is a simple AI Voice Assistant built using Python. It can interact with the user, respond to commands, and perform basic tasks like searching information, opening websites, telling time, and generating jokes.

The assistant currently takes text input from the user and provides voice + text output.

🚀 Features
Converts text to speech
Searches information from Wikipedia
Opens websites like YouTube and Google
Tells current time
Generates random jokes
Greets user based on time of day
Continuous interaction until exit
🛠️ Technologies Used
Python
SpeechRecognition
pyttsx3
Wikipedia API
pyjokes
datetime
webbrowser
📦 Installation
Step 1: Clone the repository
git clone https://github.com/ARCHANA-01shet/Personal-AI-voice-assistant.git
cd Personal-AI-voice-assistant
Step 2: Install required libraries
pip install SpeechRecognition pyttsx3 wikipedia pyjokes
OR

python3 -m pip install SpeechRecognition pyttsx3 wikipedia pyjokes
▶️ How to Run
python3 app.py
💡 Usage
Type commands such as:

wikipedia python
open youtube
open google
time
joke
exit or bye
⚠️ Limitations
Uses text input instead of voice input
Requires internet for Wikipedia search
Speech output may not work in some environments (like browser-based IDEs)
